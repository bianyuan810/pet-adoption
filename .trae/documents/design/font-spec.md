# 字体规范文档

## 设计理念
采用清晰、易读的字体，确保在各种设备上都有良好的显示效果，同时保持设计的温馨和友好感。

## 字体系统

### 1. 字体家族
| 字体类型 | 字体名称 | 备份字体 | 用途 |
|---------|---------|---------|------|
| 主要字体 | Inter | Arial, Helvetica, sans-serif | 正文文本、标题 |
| 等宽字体 | Fira Code | Consolas, Monaco, 'Courier New' | 代码、等宽文本 |

### 2. 字号规范
| 字号级别 | 像素值 | 用途 |
|---------|-------|------|
| 标题 1 | 36px | 页面主标题 |
| 标题 2 | 28px | 页面副标题 |
| 标题 3 | 24px | 区域标题 |
| 标题 4 | 20px | 卡片标题 |
| 正文 | 16px | 主要内容文本 |
| 小正文 | 14px | 辅助文本、说明文字 |
| 微型文本 | 12px | 标签、提示文字 |

### 3. 行高规范
| 文本类型 | 行高 | 用途 |
|---------|------|------|
| 标题 | 1.2 | 各种标题 |
| 正文 | 1.5 | 主要内容文本 |
| 小正文 | 1.4 | 辅助文本 |
| 微型文本 | 1.3 | 标签、提示文字 |

### 4. 字重规范
| 字重 | 用途 |
|-----|------|
| 400 (Regular) | 正文文本 |
| 500 (Medium) | 小标题、按钮文本 |
| 600 (SemiBold) | 标题、强调文本 |
| 700 (Bold) | 主标题、重要按钮 |

## 字体使用规则

1. **标题使用**：
   - 页面主标题：标题 1，字重 700
   - 页面副标题：标题 2，字重 600
   - 区域标题：标题 3，字重 600
   - 卡片标题：标题 4，字重 600

2. **正文使用**：
   - 主要内容：正文，字重 400
   - 辅助说明：小正文，字重 400
   - 标签提示：微型文本，字重 500

3. **按钮文本**：
   - 主要按钮：小正文，字重 600
   - 次要按钮：小正文，字重 500

4. **链接文本**：
   - 主要链接：正文，字重 500
   - 次要链接：小正文，字重 400

## 设计注意事项

1. 确保文本的可读性，避免过小的字号
2. 保持字体使用的一致性，相同类型的文本使用相同的字体设置
3. 合理使用字重和字号来区分不同级别的信息
4. 考虑不同设备上的显示效果，确保响应式设计

## 示例

### 标题层级
```
<h1 class="text-3xl font-bold leading-tight">页面主标题</h1>
<h2 class="text-2xl font-semibold leading-tight">页面副标题</h2>
<h3 class="text-xl font-semibold leading-tight">区域标题</h3>
<h4 class="text-lg font-semibold leading-tight">卡片标题</h4>
```

### 正文文本
```
<p class="text-base leading-relaxed">主要内容文本，行高 1.5，确保良好的可读性。</p>
<p class="text-sm leading-normal">辅助说明文字，提供额外的信息。</p>
<span class="text-xs font-medium">标签或提示文字，用于标识或说明。</span>
```
