- [ ] 状态码（code）规范
ode	HTTP 状态码	含义	适用场景
200	200	成功	查询 / 更新 / 删除操作成功
201	201	创建成功	新增资源（POST）
400	400	客户端参数错误	缺必填字段 / 参数格式错误 / 校验失败
401	401	未授权 / Token 过期	未登录 / Token 无效 / Token 过期
403	403	禁止访问 / 权限不足	登录后无操作权限
404	404	资源不存在	查询的资源（用户 / 文章）不存在
409	409	资源冲突	重复创建（如邮箱已存在）
429	429	请求过于频繁	接口限流触发
500	500	服务器内部错误	数据库异常 / 第三方接口调用失败等未知错误
